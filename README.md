## 🚀 Getting Started & Git LFS Setup

This repository uses **Git LFS (Large File Storage)** to handle large assets (3D models, textures, audio). You must install Git LFS before cloning the project, or it will be missing core assets.

### 1. Install Git LFS
* **macOS:** Open Terminal and run `brew install git-lfs` (Requires Homebrew).
* **Windows:** Download and run the installer from [git-lfs.com](https://git-lfs.com/).

### 2. Initialize LFS
Once installed, open your terminal and run this command once to register it:
```bash
git lfs install
